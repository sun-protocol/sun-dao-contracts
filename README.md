# sun-dao-contracts
Vyper contracts used in the Sun Governance DAO.

## Contract

`veSUN`: vote-escrowed SUN token for SUN DAO as equity certificate

`Gauge`: Farm pool for staking sun or other assets

`GaugeController`: controller contract for Gauge

`minter`: reward mint contract,from `Distribution` contract

`Distribution`: reward distribution conttract,only minter can transfer asset
